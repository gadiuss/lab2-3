Анализ данных сенсора
ФИО: Томилов Михаил Федорович
Группа: СКБ251

1. Задание
Написать программу для анализа показаний сенсора на языке C++.

2. Описание алгоритма
  1. Основные функции программы:
     1. Функция inputData():
      принимает указатель на массив и количество элементов
      запрашивает ввод значений от пользователя
      проверяет корректность ввода 
      очищает буфер ввода при ошибках
     
     2. Функция average():
      принимает указатель на массив и количество элементов
      вычисляет среднее арифметическое всех значений
      возвращает вещественное число
     3. Функция minValue():
      принимает указатель на массив и количество элементов
      находит минимальное значение в массиве
      возвращает найденное минимальное значение
     4. Функция maxValue():
      принимает указатель на массив и количество элементов
      находит максимальное значение в массиве
      возвращает найденное максимальное значение
     5. Функция filterAboveAverage():
      принимает указатель на массив, количество элементов и ссылку на переменную для нового размера
      вычисляет среднее значение массива
      создает временный массив
      отбирает элементы, превышающие среднее значение в временный массив
      создает новый массив с отфильтрованными значениями
      освобождает временную память
      возвращает указатель на новый массив
    6. Главная функция main():
      выводит заголовок программы
      запрашивает количество измерений 
      выделяет динамическую память под массив данных
      вызывает функцию inputData() для заполнения массива, filterAboveAverage() для фильтрации данных
      вычисляет и выводит необходимую статистику 
      освобождает выделенную динамическую память
      завершает работу программы


3. Вывод
Написал программу на языке C++ для анализа данных сенсора. Вспомнил синтаксис языка C++, практиковался с динамической памятью (new/delete), использовал указатели.
